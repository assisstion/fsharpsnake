# Card Game Snake
Programming language for creating turn-based card games.

Test out my project at [this link](http://assisstion.github.io/projects/cardgamelang/).

## Prerequisites
- [.NET Core](https://www.microsoft.com/net/download)

## Usage
- Build: `dotnet build`
- Run file: `dotnet run <filename>`

## Program Features
- Parsing & Interpreting
    - Multi-step parsing
    - Input validation
    - Line number information for debugging
    - Fail-fast exceptions
- Web Client
    - Parsing
    - Interpreting
    - Printing output

## Language Features
- Function definitions
    - Entry point (main() function)
- Function calls
    - Functions with arguments
    - Recursion
- Scope blocks
    - Indentation
- Variables
    - let statement
    - Assignment
    - Lexical scoping
    - Persistent
- Control flow
    - return statement
    - if/elif/else statements
    - while statement
- Operators
    - Numeric (+, -, *, /, %)
    - Boolean (&, |)
    - Comparison (==, !=, <=, >=, <, >)
    - Unary (-, +, !)
    - Operator assignment (+=, -=, *=, /=, %=, &=, |=)
- Parenthases
- Literals
    - String (e.g. "Hello, world!")
    - Integer (e.g. 5, -12)
    - Boolean (true/false)
    - Reference (this)
- References
    - Property accessor (e.g. print(player.name))
    - Property function call  (e.g. player.turn())
    - Property assignment (e.g. player.name = "Markus")
    - Creating new instances (e.g. player())
